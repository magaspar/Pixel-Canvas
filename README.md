# Pixel Canvas

Simple 32x32 pixel drawing app built with React + TypeScript + Vite.

## Features
- 32x32 grid
- Click & drag to paint
- Color picker
- LocalStorage autosave
- Clear canvas
- Export to PNG (nearest-neighbor, crisp)

## Development
Install deps and start dev server:

```bash
npm install
npm run dev
```

## Testing
```bash
npm test
```
